# Movie Scraper

أداة لاستخراج بيانات الأفلام من موقع topcinema.media

## المميزات
- استخراج جميع الأفلام من الصفحة الرئيسية
- استخراج البيانات الكاملة لكل فيلم:
  - العنوان والصور
  - القصة والوصف
  - تقييم IMDB
  - التفاصيل (الجودة، المدة، السنة، الممثلين، إلخ)
  - سيرفرات المشاهدة
  - سيرفرات التحميل حسب الجودة
- حفظ البيانات في مجلد `movies/`
- إنشاء فهرس شامل في `index.json`
